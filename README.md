# Zaza Woods — Esstisch-Konfigurator

3D-Konfigurator für Esstische auf zazawoods.de. Eigenständig vom
**Picknick-Konfigurator** (https://github.com/zazawoods/zazawoods-picknick-konfigurator).

## Katalog

- **9 Tischformen**: Boogvorm, Danish Oval, Halfrond, Kiezel, Oval, Organic, Rectangle, Round, Verbaan
- **37 Gestelle (Tischbeine)**: 4 Legs on Pole, A shape, Base, Blok, Butterfly, Conisch, Cris Cross, Demi Lune, Diagonal Pole, Flach Stahl, Flat Dinning V, Gerond, Hairpin, Halve Plus, Hapa, Japandi, Klassiek Midden, Kolom Oval, Kolom Plus, Kolom Rod, Konische, Matrix, Oval Column, Pilaar, Pillars, Schuin, Schuin 2.5 Voet, Stative, Tapse Spin, Thore, Triple, Twist, V Shape, Vierpoot, VN, Walrus, X Modern, X Shape
- **14 Holzfinishes**: Oak Natural, Cocoa, Mist 5, Vanilla, Pure, Golden Hour, Macchiato, Shell Grey, Walnut, Charcoal, Chocolate, Deep Black, White 5%, Yakisugi

## Tech Stack

Identisch zum Picknick-Konfigurator: three.js r160, Express, Railway, AR via
iOS Quick Look + Android Scene Viewer.

## Lokal starten

```
npm install
npm start          # http://localhost:3000
```

## Deployment

Auto-deploy via Railway → main-Branch.
